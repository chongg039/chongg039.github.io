自用 Chrome 插件分享

Sat, May 21, 2016

这里会持续不断分享一些自己觉的好用的 Chrome 插件，有些也会弃用。

2016-05-21 更新

[feedly Mini]：RSS 订阅器，你可以在这里找几个前端相关的开发者博客，或者自己去 Github 搜一些

SwitchyOmega：配合 ss 使用的科学上网利器，需要相应的规则文件

OneTab：当你打开一大堆网页而不想关掉，要留在以后再看。它可以帮你一键保存，从而避免了大量后台网页占用系统资源

Postman Interceptor：Postman 的官方插件，不过现在基本用桌面级客户端了

Responsive Web Design Tester：将网页内容在不同设备呈现

CSSViewer：实时查看网页某一部分的css

Library Sniffer for Google Chrome：Sniff web framework and javascript libraries run on browsing website.

Page Load Time：分析网页具体部分的加载速度

Save To Pocket：将未看完的内容保存在 Pocket ，以后再看，多平台可用

Isometric Contributions：将 Github 上的贡献活动变为盖楼样式显示，很有意思

octotree：Extension to show code tree for GitHub and GitLab

划词翻译：字面解释，浏览文档时必需

JSONView：Validate and view JSON documents

2017-04-10 更新

ColorZilla：Advanced Eyedropper, Color Picker, Gradient Generator and other colorful goodies

ipinfo.io：ipinfo.io 官方插件，查询网页的服务器 ip 所在地等信息

掘金：掘金网站官方插件，将 New Page 变得对开发者更亲和，为设计师、程序员、产品经理每日发现优质内容

Github Repository Size：Automatically adds repository size to GitHub’s repository summary

FeHelper：WEB前端助手，包含一些比较实用的工具
